## 自然语言编辑

直接用口头描述修改，无需菜单操作：

- "把第三页改成案例分析"
- "把这个图换成饼图"
- "背景调暗一点"
- "加一个要点列表"

## 大纲编辑

在大纲编辑器中输入指令调整结构：

- "在引言后面加一节竞品分析"
- "合并最后两页"
- "把总结移到问答页前面"

也可以手动拖拽页面重新排序，或编辑章节归属。

## 页面属性面板

预览页右边缘常驻一个拉手，点开就是当前页的属性面板。所有字段都能就地修改并自动保存（停止输入约 1 秒后写回，面板顶部会显示「保存中 / 已保存」），不用来回开关弹窗：

| 字段 | 说明 |
|------|------|
| 标题 | 页面标题，同步显示在左侧缩略图列表 |
| 所属章节 | 页面归属的章节名，留空表示未分组 |
| 页面描述 | 生成图片的依据，和详细编辑器里是同一份内容，支持 Markdown、粘贴图片和从素材库插图 |
| 描述额外字段 | 视觉元素、视觉焦点、排版布局、演讲者备注等，字段列表跟随详细编辑器的「描述设置 → 额外字段」；标了「不影响图片生成」的字段不会进入生图提示词 |
| 旁白讲稿 | 折叠在面板最底部，导出讲解视频时朗读；留空则由导出流程自动生成 |

每页独立模板模式下还会出现「模板」区块，可以直接给这一页换模板图、写模板提示词，也能跳转到批量配置。面板底部还会展示该页的状态、页码、图片版本数和创建/更新时间。

拖动面板左边缘可调整宽度（300–640px），双击边缘恢复默认宽度，也可以聚焦边缘后用 ←/→ 微调、Home/End 直接到两端。宽度和展开状态会记住，下次打开预览页保持一致。窄屏上面板改为浮层显示，点击遮罩即可关闭。

<Note>
描述改动只会更新文字，不会自动重画图片。想让改动生效，记得点「重新生成」。
</Note>

<Tip>
页面的「要点」属于大纲阶段，只在大纲编辑器里出现；到了预览页，真正驱动出图的是页面描述。
</Tip>

## 就地编辑与区域选图

在预览页点击「**编辑**」，当前页会上移、下方展开指令区，让你边看大图边改，不再弹窗遮住幻灯片（窄屏空间不足时仍改为弹窗）：

1. 在「输入修改指令」里描述要改什么（例如：把背景改成蓝色、增大标题字号）
2. 可选：点图片左上角的「**区域选图**」，直接在大图上拖拽框选——框中的部分会裁剪下来作为参考图一起提交，帮 AI 聚焦到你要改的地方
3. 可选：从素材库选择或上传更多参考图
4. 点击「**生成图片**」，AI 结合页面描述、指令与参考图重绘该页

<Note>
区域选图框出的部分是作为**参考图**提交的，用来告诉 AI 你关心哪一块，并不是只重画框内、框外像素级保留。想精确擦除或局部替换某个元素，可用顶部「素材生成」里的局部编辑与智能擦除工具。
</Note>

## 描述详细程度

在详细编辑器的「批量生成描述」按钮旁，可选择描述的详细程度：

| 级别 | 说明 |
|------|------|
| 精简 | 简洁描述，生成速度更快，适合快速原型 |
| 默认 | 平衡详细程度与生成速度 |
| 详细 | 更丰富的描述，AI 有更多构图参考，适合精细打磨 |

## 历史版本

每次重新生成幻灯片图片时，旧版本会自动保留。在预览页点击「历史版本」可查看和切换到任意历史版本，不必担心误操作覆盖满意的效果。
